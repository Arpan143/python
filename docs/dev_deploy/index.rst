开发和部署
===========================================

Python 发布应用：

    #. 以脚本发布，适用于较小的项目
    #. 以 Python 安装包发布
    #. 以 X As a Service 的方式，发布服务

Python 开发和部署一般使用独立虚拟的 Python 环境，这样使得所有开发者使用相同的环境做开发，使得开发和部署的环境尽可能一致。
