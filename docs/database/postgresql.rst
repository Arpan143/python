Solution

- use database

    根据[stackoverflow](https://stackoverflow.com/questions/10335561/use-database-name-command-in-postgresql), 可知 postgresql 只有重新建立与数据库的连接，才能切换数据库。

- 在 postgresql 中单引号和双引号

    postgresql中字符串需要用单引号包裹，如果使用双引号，系统将认为那是变量。
